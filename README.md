# AdventureWorks Sales Analytics Dashboard

## Overview
This project is an interactive Power BI dashboard developed using the AdventureWorks dataset to analyze sales performance, customer behavior, and product profitability.

## Tools Used
- Power BI
- DAX
- Data Modeling

## Key Features
- Revenue, Profit, Orders, and Return Rate KPIs
- Customer Analysis
- Product Performance Analysis
- Sales Trends
- Interactive Filters
